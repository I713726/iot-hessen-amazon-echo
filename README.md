# iot-hessen-amazon-echo

SourceCode for the talk at IoT Hessen 2017: Amazon Echo 

## Prerequisites

1. JDK 8

## Running 

1. Run `mvnw spring-boot:run`

## Building

1. Run `mvnw clean package`
1. See `target` folder for fat jar
